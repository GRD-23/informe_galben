Este documento registra los prompts utilizados con herramientas de Inteligencia Artificial (ChatGPT y GitHub Copilot) durante el desarrollo del proyecto, junto con las correcciones aplicadas y una reflexión final.

Se utilizó para elaborar el archivo `01_resumen_galben.md`.  
**Corrección:**  
Se ajustó el lenguaje para hacerlo más académico y se eliminaron frases redundantes.

## Reflexión final

El uso de IA fue un apoyo clave para estructurar y redactar el informe. La IA permitió generar borradores rápidos y organizados, pero fue necesario aplicar correcciones para ajustar el lenguaje, citar normas específicas y cumplir con la rúbrica.  

La diferencia entre usar IA como **agente programador** (Copilot en VS Code) y como **chatbot externo** (ChatGPT) fue clara:  
- Copilot resultó más eficiente para integrar código y componentes React.  
- ChatGPT fue más útil para redactar y analizar contenido académico.  

En conclusión, la IA funcionó como **herramienta de apoyo**, no como sustituto del trabajo intelectual. El valor estuvo en dirigirla con prompts específicos y validar cada respuesta con criterio crítico.
